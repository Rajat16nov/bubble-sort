### Suggest effective image formats for different purposes

Choosing the right image format can make a big difference in visual quality and file size.

For high-quality images with detailed graphics and no loss of quality, use TIFF or RAW.

If you need transparency in your image, PNG is the best option.

For web images where file size matters more than quality, use JPEG.

<strong>ChatGPT Prompt:</strong>

```plaintext

1.

High-quality print graphics

2.

Web graphics with a smaller file size

3.

Images requiring transparency
```

### Explain the role of image alt text in SEO

Image alt text plays a crucial role in SEO as it provides a textual description of an image to search engine crawlers, enhancing the understanding of your webpage's content.

It boosts accessibility for visually impaired users who use screen readers.

Alt text can also appear in place of an image if it fails to load, ensuring that the context and message are not lost.

Therefore, including relevant keywords in your alt text can help improve your site's SEO performance.

<strong>ChatGPT Prompt:</strong>

```plaintext

Provide practical tips on how to effectively utilize this feature to improve website ranking.
```
